2. Rules

The last line can be adjoint if the previous are in the proof

2.1. →I

i  | |i A     ass
j  | |i B
j+1| A→B     →I i-j

2.2. →E

i  | A→B
j  | | A
l  | | | B   →E i,j

or

i | A
j | | A→B
l | | | B    →E j,i

2.3. ∧I

i | A
j | | B
l | | | A∧B    ∧I i,j

or

i | A
j | | B
l | | | B∧A    ∧I j,i

2.4. ∧E

i | A∧B
j | | A    ∧El i

or

i | A∧B
j | | B    ∧Er i

2.5. ∨I

i | A
j | | A∨B    ∨I i,B

or

i | A
j | | B∨A    ∨I B,i

2.6. ∨E

a | A∨B
b | |b A    ass
d | |b C
k | |k B    ass
h | |k C
p | C     ∨E a,b-d,k-h

or

a | A∨B
b | |b B    ass
d | |b C
k | |k A    ass
h | |k C
p | C     ∨E a,k-h,b-d

2.7. ↔I

b | |b A    ass
d | |b B
k | |k B    ass
h | |k A
p | A↔B     ↔I b-d,k-h

or

b | |b A    ass
d | |b B
k | |k B    ass
h | |k A
p | B↔A     ↔I k-h,b-d

2.8. ↔E

b | A
d | | A↔B
p | | | B     ↔El d,b

or

b | A↔B
d | | A
p | | | B     ↔El b,d

or

b | A↔B
d | | B
p | | | A     ↔Er b,d

or

b | B
d | | A↔B
p | | | A     ↔Er d,b

2.9. ¬I

b | |b A
d | |b ⊥
d+1| ¬A     ¬I b-d

2.10. ¬E

b | ¬A
d | | A
d | | | ⊥     ¬E b,d

or

b | A
d | | ¬A
d | | | ⊥     ¬E d,b
