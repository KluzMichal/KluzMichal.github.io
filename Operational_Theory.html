There is a set \(E=\{e_1,..., e_{n_E} \}\) of reports of some conducted experiments. Obviously this set is always finite. We aim to form a theory describing this set of experimental reports.

  1. The description at the zeroth level of abstraction is simply this set of experimental reports itself. However this description is not too useful, does not allow us to make predictions about future experiments. To make predictions we have to introduce some further structure. Here is one possible way:
  2. As a first abstraction we introduce a set \(EP\) whose elements we'll call "experimental procedures" and a surjection \(T_{EP}:E\to EP\), which assigns to every experimental report an experimental procedure which this experiment in the report is supposed to be a realization of. We say the the experimental report \(e_i\) reports an experiment which realized experimental procedure \(T_{EP}(e_i)\), or simply an experiment of type \(T_{EP}(e_i)\). \(T_{EP}\) is a surjection, which means that for every experimental procedure \(ep\in EP\) there is at least one experiment in our data which is supposed to realize that procedure. Experimental procedure \(ep\in EP\) is some list of steps for the experimentalist to perform and a collection of physical objects (physical bodies, devices etc.) allowing them to perform those steps.
  3. Now we introduce a set sentences \(Sent\) and a function \(Q:Sent\times E\ni (s,e)\to q(s,e)\in\{0,1\}\) such that \(q(s,e)=1\) iff sentence "In experimental report e, s" is true. For example if \(s\in Sent\) reads: "This and this measurement device gave this and this reading" then \(q(s,e)=1\) iff in experiment \(e\in E\) this and this device gave this and this reading. If they didn't record using that device in experimental record \(e\), or they recoeded that it gave a different reading, then \(q(s,e)=0\).
  4. We further introduce a function \(O:E\to \mathbb{Q}^{n}\) This allows us to define a function

  1. At the next level of description one could make a function \(O:E\to X\) for some set \(X\), assigning to every experimental report \(e_i\in E\) its outcome \(O(e_i)\in X\). To allow for any mathematical description, those outcomes should be some mathematical objects. Now the most general outcome would be an $n$-tuple of real numbers. It should be stated what does every number stand for in a given experimental record. Now since the measurement accuracy is always finite, all those real numbers will be in fact rational numbers. So the outcome function is \(O:E\ni e_i\to O(e_i)\in Q^{n_i}\).
